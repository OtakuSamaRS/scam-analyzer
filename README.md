# 🛡️ Scam Analyzer

A powerful web-based tool designed to help detect, analyze, and protect against common scams and fraudulent activities.

## Features

- **Real-time Scam Detection** - Analyze suspicious messages, emails, and links
- **Pattern Recognition** - Identify common scam patterns and red flags
- **Educational Resources** - Learn about different types of scams
- **Interactive Interface** - User-friendly design with dark mode support
- **Fast & Responsive** - Instant analysis results

## Demo

Visit the live website: [Scam Analyzer](https://otakusamars.github.io/scam-analyzer)

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Python (Flask/FastAPI for API support)
- **Styling**: Custom CSS with dark mode support

## Installation

1. Clone the repository:
```bash
git clone https://github.com/OtakuSamaRS/scam-analyzer.git
cd scam-analyzer
```

2. Install Python dependencies:
```bash
pip install -r requirements.txt
```

3. Run locally:
```bash
python scam_analyzer.py
```

4. Open `index.html` in your browser or access the web interface

## Usage

1. Enter the suspicious text, email, or link you want to analyze
2. Click "Analyze" to scan for scam patterns
3. Review the analysis results and recommendations
4. Get tips on how to protect yourself
